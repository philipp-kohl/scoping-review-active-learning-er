## Screening Process Documentation

This table is very large. Thus, we recommend to use the [TSV file](screening_data.tsv).
The last row of the table shows sums of specific columns which can help to comprehend and reproduce the Figure 2's numbers.

*Combination* is composed of the following exclusion reasons:
- semi supervised learning
- reinforcement learning
- weak supervision
- self learning
- pre-tagging
- distant supervision
- adversial
- self training
- proactive learning
- data-augmentation
- over-labeling
- imitation learning

*Other* is composed of the following exclusion reasons:
- not-reproducible
- AL strategy uses ML
- feature AL
- Explainable AL
